JavaScript

JavaScript is a cross-platform, object-oriented scripting language. It uses objects such as arrays, date, and math. It is similar to Java, since it follows most of its expression syntax, naming conventions and basic control-flow constructs which is why it was renamed from LiveScript to JavaScript. However, JavaScript is fundamentally different from Java. You donâ€™t have to declare all variables, classes, and methods.

JavaScript is standardized by the ECMA (European Computer Manufacturers Association) to provide a standardized, international programming language based on JavaScript.

JavaScript is split into 2 languages: Client-side and Server-side.

Client-side JavaScript supply objects to control a browser and its document object model (DOM)
Server-side JavaScript supply objects related to running JavaScript on a server

In JavaScript, I learned how display words and averages, upload files while displaying the files in a separate webpage, and create a contact form using SQL database to connect to my email. Also I was even able to create a quadratic equation using a function to solve it.
