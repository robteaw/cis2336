PHP (Hypertext Preprocessor)

PHP is a programming language for making dynamic and interactive web sites. Since PHP is a programming language, it uses various programming codes such as arrays, if, and else statements. It can be used in conjunction with HTML within the same file.

In PHP, I learned to make forms, calculator, and attach the header and footer through PHP. In order to attach a header or footer, it must be created in a separate file save as php and in the main file will have “include” and name of php file. In addition, there are various forms that I have learned to make that are basically relative to today use of forms. Social security forms, address forms, and transaction forms are the types of forms learned in this class.

There are a number of types of methods used in PHP such as request, post, and get.

Request collect data after submitting html forms
Post transfers information via HTTP headers
Get sends the encoded user information appended to the page request
