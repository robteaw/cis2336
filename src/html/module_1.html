HTML (Hypertext Markup Language)

Module 1 covers HTML and CSS. Most of the lessons comes from Codeacademy. The assignments in this module taught me how to create the basis of my website.

HTML is the standard markup language that provides web sites and web applications the structure it needs. It is represented by tags and attributes. In HTML, I learned how to add images, lists, links, tables, divs, and spans to my web site.

CSS (Cascading Style Sheets)

CSS is a style language used to describe the appearance and formatting of your HTML. It can control the layout of multiple websites. CSS took HTML to another level by allowing me to change things like the font, image size, background color, margin, border, and padding. It provided me with a way to organize its structure while managing the aesthetic of the web site. Also I was able to save time by using the same css code to manage all my module websites.
