MySQL

MySQL is an open source database application. Its name is a combination of My, the name of the co-founder Michael Widenius' daughter, and SQL. It is used for developing web-based software applications. It can be used with many languages including PHP, PERL, C, C++, JAVA, etc.

SQL (Structured Query Language)

SQL is the standard language for accessing databases. It uses declarative statements to keep data accurate and secure.

In SQL, I learned to create tables and column, sort, view, store, and delete information within the database. I learned to connect a PHP form to MySQL database and create a table from the inputs. Below is some of the things I learned to use within my database by using SQL.

CREATE used to create table, rows, or database
ALTER used to change a table or row
DROP used to delete an entire table
SELECT used to select specific information
SHOW used to list records or information
INSERT used to insert new records into a table
